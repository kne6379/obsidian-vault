# 라스쿨

## 개요

KB금융그룹과 협업하여 소외계층 학생들의 교육을 지원하기 위한 학습 관리 플랫폼입니다. AI 강의 요약, 디지털 교재, 진단 시험, 멘토-멘티 학습 관리 기능을 제공하며, 학생들의 종합적인 학습 성과를 대시보드로 시각화합니다.

**기간**: 2025.02 ~ 현재
**역할**: 데이터 설계 + 백엔드 API 개발 (프론트엔드는 Bubble 노코드)
**협업**: KB금융그룹

---

## 기술 스택

| 분류       | 기술                                 |
| -------- | ---------------------------------- |
| Frontend | Bubble (노코드 플랫폼)                   |
| Backend  | Bubble Backend Workflow (서버리스 API) |
| Database | Bubble Database, Data API          |
| Logging  | Slack Integration, Bubble Database |
| AI/LLM   | LLM (데이터 교정용)                      |

---

## 서비스 주요 기능

| 기능 | 설명 |
|------|------|
| AI 강의 요약 | AI 기반 강의 내용 자동 요약 |
| 디지털 교재 | 소단원-중단원-대단원 계층 구조의 학습 콘텐츠 |
| 진단 시험 | 300명 이상 동시 접속 실시간 시험 |
| 멘토-멘티 학습 관리 | 강사-멘토-멘티 역할 기반 학습 관리 |
| 학습 대시보드 | 진단시험 결과 + 학습 기록 기반 종합 성적 시각화 |

---

## 내가 담당한 기능

### 1. 계정 관리 시스템

- 회원가입 기능 구현
- 인증 시스템 구현
- 로그아웃 처리
- 유저 관리 (역할별 권한 체계)
- 강사-멘토-멘티 역할 기반 회원 시스템

### 2. 어드민 시스템

- 디지털 교재 추가/수정 기능
- 진단시험 예약 기능
- 어드민 대시보드 구현
- 유저 CRUD (생성, 조회, 수정, 삭제)
- 소프트 딜리트 구현

### 3. 디지털 교재 관리

**데이터베이스 설계 및 구축**
- 소단원 → 중단원 → 대단원 계층 구조 설계
- 교재 데이터 정규화

**데이터 품질 관리**
- 데이터 정합성 검사
- 맞춤법 오류 등 LLM 기반 자동 교정
- 교재 데이터 준비 및 가공

**학습 기능**
- 활성 교재 데이터 표기
- 유저가 단원 선택 시 학습 활성화
- 학습 시작 기능
- 학습률 측정

### 4. 진단시험 시스템

**시험 관리**
- 진단시험 예약 기능
- 미응시자 관리
- 재응시 기능 (기존 기록 보존)

**대규모 실시간 처리**
- 300명 이상의 멘티들의 실시간 시험 기능 구현
- Data API 활용 워크로드 최적화

**안정성 확보**
- 실시간 데이터 생성 불안정성 문제 해결
- 시험 시작과 동시에 시험 데이터 사전 생성 방식 적용
- 시험 시작/종료 시간 관리로 시험 시간 내 풀이 보장

**데이터 처리**
- HTML 데이터 일괄 처리 스크립트 작성
- 2단계 진단시험 구현

### 5. 학습 대시보드

**실시간 학습 추적**
- 사용자별 실시간 강의 이수 현황 체크
- 데일리 학습 기록 추적
- 디지털 교재 진행률 모니터링

**성적 대시보드**
- 진단시험 결과 + 디지털 교재 학습 기록 통합
- 지표 기반 종합 성적 판단
- 대시보드 시각화

### 6. 유저 데이터 관리

- 전체 유저 데이터 관리
- 진단고사 점수, 스마트 노트, 오답 노트 기록
- 학습에 따른 Slack 로깅 구현

### 7. 시스템 연동

- 똑스 ↔ 라스쿨 연동 API 구현

## 아키텍처

### 사용자 역할 체계

```
강사 (Instructor)
    └── 멘토 (Mentor)
            └── 멘티 (Mentee)
```

```
진단시험 결과 ─┐
              ├──→ 학습 대시보드 ──→ 종합 성적 판단
디지털 교재 학습 기록 ─┘
```
