# 똑스

## 개요

온라인 시험 및 학습 플랫폼입니다. 문제 풀이, 기출 풀이, 오답노트, LMS 기능을 제공하며, AI 기반 문제 생성 및 학습 도구 기능을 포함합니다.

**기간**: 2024.11 ~ 현재
**역할**: 데이터 설계 + 백엔드 API 개발 (프론트엔드는 Bubble 노코드)

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | Bubble (노코드 플랫폼) |
| Backend | Bubble Backend Workflow (서버리스 API) |
| Database | Bubble Database |
| Analytics | GTM, Google Search Console |
| AI/LLM | Gemini API, OpenAI API |

---

## 아키텍처

### 문제 풀이 시스템

- 온라인 시험 문제풀이 엔진
- OMR 패널 동기화
- 채점 시스템

### LMS 시스템

- 학급 관리
- 시험지 생성 및 배포
- 학습 기록 관리

---

## 내가 담당한 기능

### 1. 온라인 모의고사 시스템

**API 및 데이터베이스**
- 모의고사 API 구현
- 문제 데이터 수집 및 정합성 검사
- 문제 데이터베이스 구축

**문제 풀이 엔진**
- 영어 듣기 평가 기능
- 수학 문제 풀이
- OMR 패널 동기화
- 채점 시스템

**학습 기능**
- 기출 풀이 기능
- 문제 저장하기
- 오답노트 기능

### 2. 학습 자료 시스템

**데이터베이스 설계 및 구축**
- 1분 요약, 5분 탐구, 용어 사전 데이터 구조 설계
- 학습 콘텐츠 데이터베이스 구축

**온라인 열람 기능**
- 학습 자료 온라인 열람 기능 구현
- 학습 로그 기록 (DB, Slack)

### 3. AI 학습 도구

**도구 개발**
- 교과 공부용 AI 학습 도구 구현
- 수행평가용 AI 학습 도구 구현
- **15개 이상 AI 도구 개발**

**프롬프트 관리**
- 프롬프트 DB 관리
- AI 문제 생성 기능

### 4. LMS (Learning Management System)

**학급 관리**
- 학급 생성 및 관리
- 학생 관리

**시험 관리**
- 시험지 생성 기능
- 시험지 배포 기능
- 스케줄링을 통한 개별 시험 기능

### 5. 기타

- GTM 설정 및 관리
- 구글 서치 콘솔 sitemap 관리
- DB 마이그레이션
- 개발 시연용 데모 페이지 개발

- 학습 자료 시스템 구축 (1분 요약, 5분 탐구, 용어 사전)
