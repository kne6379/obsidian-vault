# business 폴더 가이드

이 문서는 `business` 폴더에서 작업할 때 참조하는 **영역별 지침서**입니다.
공통 규칙(문체, 용어, 프론트매터, 링크 등)은 루트 `CLAUDE.md`를 따릅니다.

---

## 이 폴더의 목적

`business` 폴더는 **AI 기반 사업 기획과 아이디어 탐색**을 중심으로 구성되어 있습니다.

| 영역 | 주요 주제 |
|------|-----------|
| **브레인스토밍** | 시장 조사, 아이디어 발산, 평가 |
| **사업 기획** | 비즈니스 모델, 시장 분석, 경쟁 분석 |
| **실행 계획** | MVP 설계, 로드맵, 리소스 계획 |

---

## 폴더 구조

```
business/
├── CLAUDE.md            ← 현재 문서
├── _meta/               # 메타 문서
│   ├── RULES.md         # 상세 작성 규칙
│   ├── GLOSSARY.md      # 용어 정의
│   └── INDEX.md         # 전체 문서 목록
├── _templates/          # 문서 템플릿
│   └── BRAINSTORM.md    # 브레인스토밍 템플릿
└── brainstorms/         # 브레인스토밍 결과
```

---

## 작업 전 필수 확인

1. `_meta/RULES.md` - 섹션 구조, 메타데이터 형식
2. `_meta/INDEX.md` - 전체 문서 목록 (중복 방지)
3. `_meta/GLOSSARY.md` - 용어 정의
4. `_templates/BRAINSTORM.md` - 브레인스토밍 템플릿

---

## 태그 체계

**유형**: `brainstorm`, `plan`, `analysis`

**주제**:
- AI/기술: `ai`, `llm`, `saas`, `automation`
- 시장: `market`, `competitor`, `trend`
- 사업: `business-model`, `mvp`, `revenue`

**상태**: `idea`, `exploring`, `validated`, `archived`
