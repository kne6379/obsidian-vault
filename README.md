# Obsidian Knowledge Vault

Obsidian과 Claude를 활용하여 AI 기반으로 지식을 정리하고 관리하는 개인 문서 저장소입니다.

---

## 구조

```
.
├── develop/          # 개발 지식
│   ├── concepts/     # 개념/이론 문서
│   ├── _meta/        # 규칙, 용어집, 인덱스
│   └── _templates/   # 문서 템플릿
│
└── work/             # 업무 문서
    ├── projects/     # 프로젝트 설계/명세
    └── notes/        # 미팅/리서치 노트
```

## 작성 규칙

- 문체: `~입니다/~합니다` 체
- 링크: Obsidian 위키링크 `[[문서명]]` 사용
- 프론트매터: `created`, `updated`, `tags`, `status` 필수

상세 규칙은 각 폴더의 `_meta/RULES.md` 참조.
