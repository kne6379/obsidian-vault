---
description: AI 사업 브레인스토밍 (시장조사 + 아이디어 발산)
---

"$ARGUMENTS" 주제로 AI 기반 사업 아이디어를 브레인스토밍하고 문서로 정리합니다.

## 실행 순서

### 1. 주제 파싱

$ARGUMENTS 해석:
- 비어있으면: 사용자에게 주제를 질문
- 키워드가 있으면: 해당 주제로 진행

### 2. 시장 조사

웹 검색으로 다음을 조사합니다:
1. **시장 현황** - 해당 분야의 AI 적용 현황, 시장 규모
2. **주요 플레이어** - 기존 서비스/기업, 경쟁 구도
3. **트렌드** - 최신 기술 동향, 성장 방향
4. **페인 포인트** - 현재 시장에서 해결되지 않은 문제

### 3. 아이디어 발산

시장 조사를 바탕으로 사업 아이디어를 도출합니다:
1. **최소 5개 이상** 아이디어 생성
2. 각 아이디어에 대해:
   - 한 줄 요약
   - 타겟 고객
   - 핵심 가치 제안
   - 활용 AI 기술
   - 수익 모델 (간략)

### 4. 아이디어 평가

각 아이디어를 다음 기준으로 간략 평가합니다:
- **실현 가능성** (기술 난이도, 리소스 요구)
- **시장성** (수요, 경쟁 강도)
- **차별성** (기존 솔루션 대비 강점)

5점 척도로 점수를 매기고, 상위 2-3개를 추천합니다.

### 5. 문서 작성

business/brainstorms/ 하위에 문서를 생성합니다:
- 파일명: `YYYY-MM-DD $ARGUMENTS.md`
- business/_templates/BRAINSTORM.md 템플릿 기반

### 6. 인덱스 업데이트

- business/_meta/INDEX.md에 문서 추가
- 새 용어가 있으면 business/_meta/GLOSSARY.md에 추가

### 7. 관련 문서 링크

- 볼트 내 관련 문서 탐색 (develop/, work/ 포함)
- 양방향 링크 연결

### 8. Git 커밋 & 푸시

- 변경사항 커밋
- origin에 푸시

## 출력 형식

문서 저장 후, 대화에도 핵심 요약을 출력합니다:

```
## 브레인스토밍 결과: {주제}

### 시장 핵심 인사이트
- (3줄 이내)

### 추천 아이디어 TOP 3
1. **아이디어명** - 한 줄 요약 (실현/시장/차별: ★★★)
2. ...
3. ...

### 다음 단계 제안
- ...

📄 상세 문서: business/brainstorms/YYYY-MM-DD 주제.md
```

## 주의사항

- 시장 조사는 최신 정보 기반 (웹 검색 필수)
- 아이디어는 구체적이고 실행 가능한 수준으로
- 이미 존재하는 유사 브레인스토밍 문서가 있으면 참고하여 중복 방지
- 모든 단계 완료 후에만 커밋 & 푸시
