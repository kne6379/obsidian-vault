---
description: 문서 품질 검수
---

"$ARGUMENTS" 문서를 검토하고 다음 기준으로 품질을 검수합니다.

## 검수 항목

1. **프론트매터 확인**
   - created, updated, tags, status 필수 필드 존재 여부
   - 날짜 형식 (YYYY-MM-DD)
   - 태그가 GLOSSARY/RULES에 정의된 것인지

2. **문체 검수**
   - `~입니다/~합니다` 체 사용 여부
   - 비문, 오타 확인

3. **구조 검수**
   - 필수 섹션 존재 여부 (정의, 관련 문서, 참고 자료 등)
   - 빈 섹션 여부

4. **링크 검수**
   - `[[링크]]` 형식 사용 여부
   - 존재하지 않는 문서 링크 여부

5. **내용 검수**
   - 정의가 명확한지
   - 예시가 적절한지
   - 중복 내용 여부

## 출력 형식

검수 결과를 다음 형식으로 출력합니다:

```
[PASS] 통과 항목
[FAIL] 수정 필요 항목 (구체적인 수정 제안 포함)
```
