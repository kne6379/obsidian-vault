- MLOps 엔지니어 수요 폭증, 일반 ML 포지션 대비 10~15% 높은 연봉

**포지셔닝 평가:**

- **최적 포지션**: "AI 애플리케이션을 프로덕션 수준으로 구축하는 백엔드 엔지니어" — 순수 백엔드(기본기 부족 노출)나 순수 AI/ML(이론 부족)보다 교집합이 유리

#### Unlook (AI 스타일 기반 데이팅 플랫폼) — 상태: idea

기술 스택 설계(FashionSigLIP, MediaPipe, pgvector), 유저 플로우, 비즈니스 모델, 12주 타임라인까지 사실상 프로젝트 기획서 수준의 완성도를 갖추고 있습니다. 글로벌 온라인 데이팅 시장은 2026년 97.8억 달러 규모, CAGR 4.81%로 안정적 성장세를 유지하고 있습니다.

**유효성: 여전히 유효하나 차별화 포인트 재점검 필요**
- "양에서 질로의 전환" 트렌드와 "스와이프 피로 제거 + 스타일 기반 매칭" 가치 제안이 정확히 맞닿아 있습니다.
- 그러나 Overtone(음성 기반), Ditto(대학생 특화, 시드 920만 달러) 등 "얼굴 없는 매칭" 니치 플레이어가 속속 등장하여 컨셉 독점성이 약화되었습니다.
- FashionSigLIP/CLIP 기반 패션 매칭을 데이팅에 적용한 직접 경쟁자는 미확인 — 기술적 차별점은 유지됩니다.
- **최대 리스크**: 콜드스타트 문제. 2인 팀으로 양면 마켓 부트스트랩은 근본적으로 자본 집약적입니다.

#### AI 콘텐츠 생성 브레인스토밍 — 상태: idea (6개 아이디어 도출, 평가 완료)

**버티컬 AI 콘텐츠 SaaS (13/15) — 유효성: 상향**
- 2026년 AI 콘텐츠 시장의 핵심 트렌드가 "범용에서 도메인 특화로"의 전환이며, 규제 산업(의료/법률)은 범용 도구의 진입이 구조적으로 어렵습니다.
- 기술 스택(RAG, LLM, 백엔드)과 가장 잘 맞고, 고객의 지불 의사가 명확합니다.

**AI 콘텐츠 QA 플랫폼 (13/15) — 유효성: 대폭 하향**
- 이 아이디어가 가장 큰 환경 변화를 맞았습니다. Confident AI, Braintrust, GPTZero, Fiddler 등 VC 자금을 등에 업은 전문 플레이어가 이미 프로덕션 레벨 솔루션을 제공합니다.
- "AI 콘텐츠의 편집장" 포지셔닝은 매력적이나, 차별성 점수를 4/5에서 2/5로 재조정해야 합니다.

**n8n 자동화 에이전시 (12/15) — 유효성: 유지, 한국 시장 선점 여지**
- n8n은 2025년 매출 5배 성장, 기업 가치 25억 달러. 글로벌에서는 에이전시 포화 조짐이나, 한국 시장에서는 아직 초기 단계입니다.

**워크플로우 마켓플레이스 (11/15) — 유효성: 하향**
- n8n 공식 커뮤니티에 5,800개+ 워크플로우가 이미 공유되고 있으며, 플랫폼이 직접 흡수할 가능성이 높습니다.

knowledge 영역의 활동 패턴은 이 가치관과 상당 부분 일치합니다. 15개 문서가 전부 done 상태이며, 한 가지를 시작하면 완성하는 습관이 보입니다. "잘 읽히는 글쓰기" 문서 작성은 "말을 이쁘게 하겠다"는 가치관의 글쓰기 확장입니다. 이력서가 v1에서 v3까지 발전하고 피드백 문서까지 별도 정리한 것은 "감정이 흔들려도 행동은 유지한다"는 다짐의 실제 이행입니다.

**불일치/긴장 영역:**

"좋은 사람되기"에서 "불편한 말도 할 수 있는 용기"를 강조하면서, 가장 중요한 순간에 고백하지 못했다고 스스로 적었습니다. "용기"라는 가치가 직업적 맥락에서는 잘 발현되지만 관계적 맥락에서는 아직 어렵다는 자기 인식이 보입니다.

"떠나기" 문서가 42일째 draft입니다. 이력서는 계속 진화했는데, 정작 그 결심의 원점인 문서는 그대로입니다. 이 문서를 done으로 바꾸는 것이 무엇을 의미할지, 혹은 바꾸지 못하는 것이 무엇을 의미할지 생각해볼 여지가 있습니다.

## 5. 트렌드 브리핑

### 주목할 트렌드

**1. MCP(Model Context Protocol)와 에이전트 간 통신 표준**

2026년 2월 기준 공식 MCP 레지스트리에 6,400개 이상의 서버가 등록되었으며, Linux Foundation 산하 Agentic AI Foundation(AAIF)이 설립되어 MCP와 A2A(Agent-to-Agent Protocol)가 에이전트 시대의 HTTP에 해당하는 표준으로 자리잡고 있습니다. 볼트에는 [[CLAUDE.md 설계 원칙]]과 [[AI 네이티브 엔지니어]] 문서가 있으나, MCP/A2A 프로토콜 자체에 대한 기술 정리가 없습니다. 현재 프로젝트들에 MCP 서버를 붙이는 것만으로도 제품 가치가 올라가는 시대입니다.

**2. AI 네이티브 SaaS 아키텍처 전환**

"AI 기능이 붙은 SaaS"에서 "AI를 중심으로 설계된 SaaS"로 패러다임이 전환되고 있습니다. 추론 파이프라인, 지속적 컨텍스트 루프, 오케스트레이션 플랫폼이 핵심 아키텍처 요소가 되었습니다. ventures 영역의 버티컬 AI SaaS 아이디어가 정확히 이 흐름 위에 있습니다.

**3. 플랫폼 엔지니어링과 내부 개발자 플랫폼(IDP)**

DevOps가 티켓 기반 지원에서 셀프서비스 IDP로 진화하고 있습니다. 76%의 DevOps 팀이 CI/CD에 AI를 통합했으며, 에이전틱 AI가 인프라 관리를 자율 수행하는 방향으로 이동 중입니다. Docker, ECS, K8s 지식이 기반이 되지만 이를 통합하는 상위 개념인 플랫폼 엔지니어링에 대한 정리가 없습니다.

**4. 에이전틱 AI의 산업 전반 확산**

자율적으로 계획-실행-검증하는 에이전트 시스템이 주류로 부상했습니다. 오케스트레이터가 전문 에이전트들을 조율하는 "퍼펫티어" 패턴이 표준이 되고 있습니다. OWASP Agentic AI Top 10이 발표되고, 인증 없는 MCP 서버 492개가 발견되어 AI 보안도 핵심 이슈로 부상했습니다.

**5. AI 엔지니어 직군의 폭발적 성장**

AI 엔지니어 포지션이 전통 SW 엔지니어 대비 300% 빠르게 성장하고 있으며, AI/ML 직군이 전체 기술 직군의 50%까지 비중이 확대되었습니다. "백엔드 + AI/RAG" 조합은 시장에서 가장 수요가 높은 교차 역량에 해당합니다.

### 볼트 확장 제안

| **LLM 오케스트레이션 (LangGraph, CrewAI)** | 멀티에이전트 워크플로우 프레임워크, n8n과 결합 시 사업 실행력 강화 | [[RAG]], ventures 영역 전체 |
| **서버리스 2.0 (Durable Functions)** | 상태 유지형 서버리스, AWS Lambda 사용량 YoY 100%+ 성장 | [[ECS]], [[API Gateway]] |

### 기회 포착 포인트

1. **"백엔드 + RAG + MCP" 삼각 편대** — 현재 역량에 MCP 서버 구축만 추가하면 "AI 시스템을 설계하고 운영할 수 있는 백엔드 엔지니어"로 최적 차별화
2. **ventures 아이디어 실행 무기: MCP + 에이전틱 아키텍처** — 버티컬 AI SaaS를 MCP 서버로 구현하면 기존 AI 도구의 플러그인으로 즉시 배포 가능
3. **Unlook 차별화 레버: 엣지 추론 + 서버리스 ML** — 엣지 전처리 + 서버리스 GPU 추론 하이브리드로 속도/비용 최적화
4. **플랫폼 엔지니어링 = 1인/소규모 팀 창업의 인프라 자유도** — IDP 수준으로 끌어올리면 여러 프로젝트를 최소 운영 부담으로 관리 가능
5. **AI 거버넌스 = 버티컬 SaaS의 해자** — 규제 산업 진입 시 보안/거버넌스가 경쟁자 모방 불가능한 이중 해자

## 6. 종합 조언

야, 전체 봤다. 직설적으로 간다.

### 당장 해야 할 것

**1. 이력서 최종본 하나만 남겨**

v1, v2, v3가 전부 draft로 42일째 흩어져 있어. 어느 게 진짜야? v3를 done으로 완성하고, v1/v2는 archived 처리해. 이직 준비한다면서 이력서가 draft인 건 모순이야.

**2. 테스트/CI/CD부터 붙여**

3차 분석 때도 말했고 이번에도 같은 소리다. 네 최대 약점이야. QGen AI에 pytest 테스트 붙이고 GitHub Actions CI 파이프라인 하나 만들어. 면접에서 "테스트 어떻게 하세요?" 물으면 지금 대답할 수 있어? 없으면 이번 주에 시작해.

**3. 실무 경험 knowledge로 꺼내**

네가 제일 돈 되는 노하우를 정리 안 하고 있어. LLM 응답 안정화 패턴, 배치 처리 패턴, 클린 아키텍처 실전 — 이게 새 기술 공부보다 급해. 면접에서도, 블로그에서도, 사업에서도 다 쓸 수 있는 자산이야.

**4. AI 콘텐츠 QA 아이디어는 접어**

3차 분석 때 블루오션이라고 했는데, 8일 만에 레드오션 됐어. Confident AI, Braintrust, GPTZero가 이미 프로덕션 레벨이야. VC 자금 등에 업은 애들이랑 기능 경쟁하는 건 2인 팀 전략이 아니야. 검증의 통찰은 버티컬 SaaS에 "규정 준수 레이어"로 흡수해.

**5. "떠나기" 다시 열어**

42일째 draft야. 3차 분석 때 35일이었고, 7일 더 지났어. done으로 바꾸든, 지금의 마음 한 줄 덧붙이든, 뭐라도 해. 안 건드리고 방치하는 건 정리가 아니라 회피야.

### 이번 주/이번 달 방향

**커리어**: 테스트 + CI/CD 4주 스프린트 시작해. QGen AI에 pytest 붙이고, GitHub Actions 파이프라인 하나 세우는 게 이번 달 목표. 동시에 LLM 응답 안정화 패턴을 knowledge 문서로 정리해. 이게 네 면접 킬러 카드야.

**사업**: 버티컬 AI 콘텐츠 SaaS로 단일 집중해. 이번 주에 타겟 산업 1개 확정하고, 다음 주에 종사자 5명 인터뷰 시작. n8n 에이전시는 한국 시장에서 캐시플로우 보조용으로 1~2건 수주 시도. Unlook은 캐시플로우 확보 후 재개.

**삶**: 6일째 문서 안 썼어. 바빠서 그런 건지, 에너지가 빠진 건지. 이번 주에 life에 뭐라도 하나 적어봐. "지금 내가 열심히 하는 것들이 진짜 원해서 하는 건지, 다른 뭔가를 느끼지 않기 위해 하는 건지" — 이 질문에 대한 대답이라도.

### 전체 평가
